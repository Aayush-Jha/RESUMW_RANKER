# Viva / Interview Questions and Answers

## Q1. What is this project about?
This project is an AI-powered resume ranking system that compares resumes with a job description and ranks them based on relevance.

## Q2. Why is this useful?
It saves time for HR teams by automating the initial screening of resumes.

## Q3. What techniques are used?
The project uses NLP-based keyword extraction, TF-IDF vectorization, and cosine similarity. It also supports a trained machine learning model.

## Q4. What files can be uploaded?
The app accepts PDF or TXT files for both job descriptions and resumes.

## Q5. What is the output?
The output includes a ranked list of candidates, match percentages, matching skills, missing skills, and CSV export.

## Q6. What is the future scope?
The system can be improved with a larger dataset, better NLP models, and support for more file formats.
