# Presentation Outline

## Slide 1: Title
AI-Powered Resume Ranker

## Slide 2: Problem Statement
Recruitment teams face difficulty screening many resumes manually.

## Slide 3: Solution
An AI-based system that ranks resumes against a job description automatically.

## Slide 4: Features
- file upload
- resume ranking
- match percentage
- skill comparison
- CSV export
- trained model support

## Slide 5: Workflow
Upload JD -> Extract text -> Process resumes -> Rank -> Display results

## Slide 6: Technologies
Python, Flask, scikit-learn, spaCy, pdfplumber, pandas

## Slide 7: Dataset and Model
Uses uploaded dataset folder and a training CSV for model experimentation.

## Slide 8: Demo
Show the web app and ranking results.

## Slide 9: Conclusion
Efficient, scalable, and useful for HR automation.
